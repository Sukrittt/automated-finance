# UI Guidelines (MVP)

## Product Feel

- User-friendly
- Playful but not noisy
- Sleek and simple
- Readable for all age groups

## Theme Direction

- Light-first interface
- Soft cards with gentle depth
- Pastel accent palette
- Clear spacing and hierarchy

## Information Architecture

- Layered summary:
  - first view: key spend totals and trend signal
  - second layer: category and period breakdown
  - third layer: transaction-level detail

## Core Screens

- Home summary
- Transactions + recategorization
- Insights
- Budgets
- Settings

## Interaction Rules

- Single-tap category correction
- Clear empty/loading/error states
- Avoid dense dashboards by default

## Accessibility Baseline

- High contrast text/background combinations
- Large tap targets
- Clear labels with icon support
- Avoid ambiguous color-only status encoding
