# Master Plan

## Objective
Build a Play-compliant, India-focused Android app that automatically tracks UPI spending from notifications and delivers clear weekly insights.

## Milestones
- M0 Foundation + Architecture
- M1 Notification ingestion
- M2 Parsing + dedupe + categorization + review queue
- M3 Sync + dashboard + AI insights
- M4 Compliance hardening + beta launch
- U0 UI research/spec
- U1 Tokenized design system + core screens
- U2 Data viz wrappers and fallback states

## Success Metrics
- Primary: auto-capture accuracy
- Secondary: weekly retention, review completion rate, crash-free sessions

## Constraints
- Android only
- English only for v1
- White theme (mono minimal)
- Play-safe permission policy (no READ_SMS in Play track)
