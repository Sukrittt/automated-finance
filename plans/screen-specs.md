# Screen Specs (v1)

## Onboarding
- Goal: explain zero-manual-entry model and permissions clearly.
- Sections: value proposition, permission summary, continue CTA.
- States: initial, permission denied guidance (to be wired in M1).

## Home Dashboard
- Goal: instant spend visibility.
- Blocks: total spend/txn stats, daily spend bar chart, category split card.
- States: no data card + permission prompt.

## Transactions Feed
- Goal: chronological verification and browsing.
- Blocks: category filters, transaction rows.
- States: empty list message, parse error tag (future).

## Review Queue
- Goal: fix low-confidence parses quickly.
- Blocks: queue cards, accept/edit actions, edit sheet.
- States: empty queue celebration state.

## Insights Weekly
- Goal: AI summary + top leak + improvement action.
- Blocks: summary, leak callout, trend line, recommendation.
- States: no insights generated yet.

## Settings/Privacy
- Goal: trust and control.
- Blocks: permission status, data controls, subscription status.
- Actions: export, delete, manage subscription.
